Instructions for Process Based Estimation tool

The following instructions will list the different ways that your able to use and interact with the PBE tiny tool

To get started with the Process based Calculator Enter a function name and an activity name as your starting points.
Select the "Add Activity" button to add more activities and select the "Add Function" button to add more functions.

Once this is done, enter the corresponding hours.

To remove a function from the calculator select the trash can Icon next to the function you want to remove

To calculate your function point total select the "Calculate" button

To refresh your table select the "Refresh" button
